# Roadmap

## Version 0.1

Desktop foundation:

- WPF app
- keyboard list
- device detection by key press
- saved keyboard rules
- tray support
- start with Windows
- GitHub Actions executable build

## Version 0.2

Improve desktop utility experience:

- better device labels
- rule enable/disable toggle in table
- import/export settings
- restore defaults button
- app icon
- first-run setup guide

## Version 0.3

Background runner:

- Windows service project
- service installer
- service reads saved rules
- service health status in desktop app

## Version 0.4

Enforcement prototype:

- prototype low-level input handling
- validate laptop keyboard Space key use case
- keep external keyboard Space key working

## Version 1.0

Production-grade enforcement:

- signed keyboard filter driver
- emergency reset shortcut
- safe mode recovery flow
- installer/uninstaller
- release packaging
